# Olivia

# Installation instruction for windows

1. Install Python.
1. run following commands in the cmd.
```
pip install speechRecognition
pip install pyttsx3
pip install wikipedia
pip install pipwin
pipwin install pyaudio
pip install -U autopep8
```
1. run the py file in any IDE.
